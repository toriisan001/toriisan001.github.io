    以下は、ExcelのテーブルであるListObjectに対して、各行の「単価」列と「数量」列の値を読み取り、
    その積を計算して「金額」列に自動的に書き込むVBAの例です。
    
    VBA
    Copy code
    Sub UpdatePrice()
        Dim tbl As ListObject
        Dim row As ListRow
        
        Set tbl = ActiveSheet.ListObjects("Table1")
        
        For Each row In tbl.ListRows
            Dim unit_price As Double
            Dim quantity As Double
            Dim price As Double
            
            unit_price = row.Range(1, "UnitPrice").Value
            quantity = row.Range(1, "Quantity").Value
            price = unit_price * quantity
            
            row.Range(1, "Price").Value = price
        Next row
    End Sub
    このVBAを実行すると、ListObject「Table1」の「単価」列と「数量」列の各行において、
