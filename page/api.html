Google YouTube Data API

環境(20230305時点)

【参考】YouTube Data API v3 を使って YouTube 動画を検索する

Google API Consoleにログイン。(グーグルアカウントを事前に作成しておく)

【参考】Google API Console

プロジェクトの作成をクリック

プロジェクト名を記入し、作成をクリック

APIとサービスの有効化をクリック

APIライブラリで「YouTube Data API」を検索してクリック

認証情報タブで「認証情報を作成」をクリック、プルダウンから「APIキー」を選択

APIキーを確認。閉じるをクリック。

作成完了。「鍵を確認します」を押すと再度確認可能。

「APIキーを編集」をクリック。

「名前」を「for youtube data api v3」に変更。キーの制限を「YouTube Data API v3」に設定。保存をクリック。
