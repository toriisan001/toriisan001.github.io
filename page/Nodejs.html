Nodejs

用語

Nodejs

クロスプラットフォームに対応したフリーでオープンソースのJavaScript実行環境

JavaScriptとの違い

JavaScript:プログラミング言語そのもの、Node.js:JavaScriptをサーバーサイドで動作させるためのランタイム環境

ランタイム環境(Runtime Environment)

プログラムが実行されるために必要なソフトウェアやライブラリ、設定

Express

Node.js 上で動く Webアプリケーションフレームワーク

NestJS

大規模サービス用フレームワーク

EJSファイル(Effective JavaScript Templating)

拡張子は「.ejs」

HTMLの中にJavaScriptのコードを直接記述し、サーバーサイドやビルド時に動的なHTMLを生成するためのテンプレートエンジン

ポート3000番

Node.js のチュートリアルやサンプルで 3000番 が使われてきた。「他のサービスとぶつかりにくい番号」として慣習化

MVC

M: Model、V: View、C: Controller

MVC とは、アプリケーションを 3つの役割に分けて設計するパターン のこと

Sequelize（シーキュライズ）

Node.js 用の ORM（Object-Relational Mapping）ライブラリ

ORM（Object-Relational Mapping）

DB の テーブルとレコードをオブジェクトとして扱える仕組み

DBをオブジェクトとして扱える便利な仕組み

基本的なDB操作はORM、複雑なDB操作はORMでは対応できないので普通にSQLで書くのが良いとされている。


■ 普通のSQL（生SQL）
SELECT * FROM users WHERE id = 1;

■ ORM（Object-Relational Mapping）
const user = await User.findByPk(1);

よく使うnpmコマンド

npm初期化。(デフォルト)

コマンド>npm install ${ライブラリ名}@latest

特定のライブラリを明示的に特定のバージョンのライブラリをインストール。

コマンド>npm view ${ライブラリ名} versions

例)npm view express versions

■ C:\xampp\apache\conf\extra\httpd-vhosts.conf
<VirtualHost *:80>
    ServerName node-app.local

    ProxyPreserveHost On
    ProxyPass / http://127.0.0.1:3000/
    ProxyPassReverse / http://127.0.0.1:3000/

    ErrorLog "logs/node-app-error.log"
    CustomLog "logs/node-app-access.log" common
</VirtualHost>

■ server.js
const http = require('http');
const fs = require('fs');
const path = require('path');

const PORT = 3000;

const server = http.createServer((req, res) => {
  console.log(`Request received: ${req.url}`);

  if (req.url === '/') {
    const filePath = path.join(__dirname, 'public', 'index.html');

    fs.readFile(filePath, (err, data) => {
      if (err) {
        res.writeHead(500);
        res.end('Server Error');
        return;
      }

      res.writeHead(200, { 'Content-Type': 'text/html; charset=UTF-8' });
      res.end(data);
    });

  } else if (req.url === '/api') {
    res.writeHead(200, { 'Content-Type': 'application/json; charset=UTF-8' });
    res.end(JSON.stringify({ message: 'JSON response from Node.js' }));

  } else {
    res.writeHead(404);
    res.end('Not Found');
  }
});

server.listen(PORT, '127.0.0.1', () => {
  console.log(`Server running at http://127.0.0.1:${PORT}`);
});

const express = require("express");
const path = require("path");
const sequelize = require("./config/database");
const pageRoutes = require("./routes/pages");

const app = express();

app.use(express.json());

// EJSを使う設定
app.set("view engine", "ejs");
app.set("views", path.join(__dirname, "views"));

// ルート
app.use("/", pageRoutes);

// 静的ファイル
app.use(express.static(path.join(__dirname, "public")));

async function start() {
  try {
    await sequelize.authenticate();
    console.log("DB connected:", sequelize.getDatabaseName());

    await sequelize.sync({ alter: false });

    app.listen(3000, () => {
      console.log("Server started: http://localhost:3000");
    });
  } catch (err) {
    console.error("Error:", err);
  }
}

start();
