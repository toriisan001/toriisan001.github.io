HTML

「!」を押してエンターキーで以下のひな型を入力（右下の言語モードがhtmlになっていることを確認）

    
    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    </head>
    <body>
        
    </body>
    </html>
    
    ・langはjaに変更する

※以下幅をデバイスの幅を合わせるという意味。

<meta http-equiv="X-UA-Compatible" content="IE=edge">

ローカル開発環境を作成

http-serverインストール

https://nodejs.org/ja/download/からNodejsのインストーラをダウンロード。インストールする

「cd C:\xampp\htdocs\MyPortfolio\server」 で サーバーをインストールするディレクトリに移動

「npm install -g http-server」 で http-server をインストール

「http-server --version」 で バージョンチェック

「npx http-server -o」 で サーバー立ち上げ

ブラウザからURL「http://127.0.0.1:8080/」 でアクセス

1分ごとにホットリロードする場合

「npm install -g reload」 でパッケージをインストール

「cd C:\xampp\htdocs\MyPortfolio\server」 で サーバーをインストールするディレクトリに移動

「reload -b」 で1分ごとにホットリロードで 立ち上げ

【参考】1分でホットリロードするローカルサーバーを立てる

ブラウザからURL「http://127.0.0.1:8080/」 でアクセス

ポートを変えたい場合、「-p ${任意のポート}」を追記するといいらしい

【参考】node.js http-serverコマンドでwebサーバーを起動する
